Flutter — Obtendo dados de uma API REST e salvando localmente no Sqlite.

Fábio Jansen
Dec 5, 2019 · 5 min read

Recentemente, ao desenvolver uma aplicação em Flutter, precisei obter dados de uma Api externa e gravar localmente no Sqlite para que os dados pudessem ser acessados sem a necessidade de uma conexão à internet.

Neste artigo, irei demonstrar o uso de três bibliotecas famosas do Flutter:
dio, path_provider e sqflite.

Nota: Para manter as coisas simples, não utilizaremos conceitos avançados como Bloc e/outros.
